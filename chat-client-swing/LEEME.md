# Chat Client (Swing) - Equipo 5

Cliente de escritorio desarrollado en Java Swing para interactuar con el servidor de chat.

## Características
* Interfaz gráfica intuitiva.
* Conexión automática al servidor local.
* Soporte para múltiples instancias simultáneas.

## Ejecución
Puedes ejecutarlo desde VS Code usando el botón **Run** en la clase `Main.java` o vía terminal:
```bash
mvn compile exec:java "-Dexec.mainClass=org.equipo5.client.Main"
