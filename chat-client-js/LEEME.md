# Chat Client (JavaScript) - Equipo 5

Interfaz web minimalista para el sistema de chat mediante WebSockets.

* HTML5 / CSS3 (Diseño responsivo).

## Instalación y Uso
1. Asegúrate de que el servidor Quarkus esté corriendo.
2. Abre el archivo `index.html` directamente en tu navegador o usa una extensión como **Live Server** en VS Code.
