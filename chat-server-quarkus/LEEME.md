# Chat Server (Quarkus) - Equipo 5

Servidor central de mensajería basado en WebSockets para la gestión de comunicación en tiempo real.

## Requisitos
* Java 17+
* Maven 3.8+

## Tecnologías
* **Quarkus**: Framework Java nativo.
* **Jakarta WebSocket**: Protocolo de comunicación bidireccional.

## Ejecución
1. Navega a la carpeta del proyecto.
2. Ejecuta el servidor en modo desarrollo:
   ```bash
   ./mvnw quarkus:dev
